Import the app:
```bash
bit import bitdev.react/examples/react-ssr-app
```

Configure the app plugin on your workspace:

```bash
bit use examples/react-ssr-app
```

### Run the application

You can run your application on a separate port to see it outside of the Bit workspace

```bash
bit run react-ssr-app
```
