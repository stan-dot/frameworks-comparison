description: Vite <-> React application component
labels: ['vite-app', 'app']
---

Import the app component:
```bash
bit import bitdev.react/examples/vite-app
```

Configure the app plugin on your workspace:

```bash
bit use examples/vite-app
```

### Run the application

You can run your application on a separate port to see it outside of the Bit workspace

```bash
bit run vite-app
```
